# BulletsNdGuns Game

Este es un proyecto para la asignatura Estructuras de Datos II de la carrera Ingeniería de Sistemas en la Universidad del Norte, el proyecto consiste en la elaboración de un juego de tematica libre en el que se utilizen Hilos y Sockets.

Se desarrolla un juego que mezcla el estilo de plataformas y shooter, en el que los jugadores deberán avanzar varios niveles tipo plataforma recolectando monedas y demás items, para después acabar con mini-jefes hasta llegar al jefe final que cuando es derrotado se gana el juego.
